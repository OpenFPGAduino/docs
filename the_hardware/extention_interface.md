# extention interface

![](mother_extention.png)  
This is the extension IO of the mother broad. The extension IOs are listed as follows:

1. Ethernet \(10/100Mbis\)
2. USB device \(2.0\)
3. USB host \(2.0\)
4. SD card
5. Audio 
6. S/PDIF
7. LCD interface\(in the back end\)
8. Programmable IO A\(\)
9. Programmable IO B\(\)
10. Power Supply
